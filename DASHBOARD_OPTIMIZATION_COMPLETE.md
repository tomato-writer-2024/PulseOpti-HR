## 🎯 企业真实用户场景分析

### Dashboard首页用户场景

#### 1. 老板/决策者
- **需求**: 快速了解企业整体情况、关键指标、人效数据
- **痛点**: 数据分散、难以快速获取关键信息
- **解决方案**: 关键指标卡片、数据大屏入口、PRO功能推荐

#### 2. HR管理者
- **需求**: 快速处理待办事项、访问常用功能、了解最新动态
- **痛点**: 待办事项多、功能查找困难、信息获取慢
- **解决方案**: 待办事项列表、快捷入口、最新动态

#### 3. 业务负责人
- **需求**: 快速启动招聘、查看团队绩效、访问业务支持功能
- **痛点**: 功能入口分散、操作流程繁琐
- **解决方案**: 快捷入口、PRO功能推荐、快捷返回

#### 4. 普通员工
- **需求**: 快速找到自助服务、了解公司动态、返回首页
- **痛点**: 功能入口难找、导航不清晰
- **解决方案**: 快捷入口、面包屑导航、返回首页

```tsx
        <div className="flex gap-2">
          <Button size="sm" variant="ghost" onClick={() => setShowProBanner(false)}>
            关闭
          </Button>
          <Link href="/premium">
            <Button size="sm">立即升级</Button>
          </Link>
        </div>
      </div>
    </AlertDescription>
  </Alert>
)}
```

### 2. 自动隐藏逻辑
```tsx
useEffect(() => {
  const timer = setTimeout(() => {
    setShowProBanner(false);
  }, 30000); // 30秒后隐藏

  return () => clearTimeout(timer);
}, []);
```
