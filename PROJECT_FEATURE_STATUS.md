## 📊 项目概览

### 技术架构
- **前端框架**：Next.js 16.1.1 (App Router) + React 19.2.3
- **开发语言**：TypeScript 5.9.3
- **UI 组件库**：shadcn/ui + Tailwind CSS 4
- **数据库**：PostgreSQL (Neon)
- **ORM**：Drizzle ORM
- **部署平台**：Vercel
- **AI 集成**：豆包大语言模型

### 代码规模
- **数据表数量**：100+ 表
- **API 路由**：223 个
- **前端页面**：150+ 页面
- **代码行数**：约 50,000+ 行
